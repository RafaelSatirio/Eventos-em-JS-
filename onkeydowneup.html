Mouse Out e Over

onkeydown e onkeyup são eventos de teclado em JavaScript que são acionados quando uma tecla é pressionada e solta, respectivamente. O evento onkeydown é acionado quando uma tecla é pressionada no teclado. Você pode definir uma função de callback para este evento e o navegador chamará essa função quando o evento ocorrer. A função de callback recebe um objeto Event como seu único argumento, que contém informações sobre o evento, como o código da tecla pressionada. O evento onkeyup é acionado quando uma tecla é liberada após ser pressionada. Você também pode definir uma função de callback para este evento e o navegador chamará essa função quando o evento ocorrer. A função de callback recebe um objeto Event como seu único argumento, que contém informações sobre o evento, como o código da tecla liberada.
