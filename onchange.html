A função onchange é um evento em JavaScript que é acionado quando o valor de um elemento HTML é alterado. O evento onchange é comumente usado em formulários para executar uma ação quando o usuário seleciona uma opção em um menu suspenso (select) ou altera o valor de um campo de entrada (input). Por exemplo, você pode usar o evento onchange para alertar que o conteúdo de uma página da web quando o usuário seleciona uma opção em um menu suspenso. Quando o usuário seleciona uma nova opção, a função associada ao evento onchange pode atualizar o conteúdo da página para exibir informações relevantes à nova seleção.
